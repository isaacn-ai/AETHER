Titan Cosmetic storefront scaffold uses Node.js 20, Tailwind CSS v3.4.19, and Vite v7.2.4.

Current checkout contains configuration files only (no application source directories such as `src/` or `public/`).

Available UI/component dependencies in this scaffold include Radix UI primitives, form tooling, charting, and animation libraries as listed in `package.json`.

Run locally:

- `npm install`
- `npm run dev`
- `npm run lint`
- `npm run build`
