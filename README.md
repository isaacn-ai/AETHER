# Titan Cosmetic (AETHER Template Content Migration)

This repository currently contains project configuration scaffolding for a Vite + React + TypeScript storefront and has been rebranded for **Titan Cosmetic**.

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

## Notes

- Branding and project-facing text in repository root files were migrated to Titan Cosmetic.
- Full in-app page/content migration requires the missing `src/` application files to be present in this repository.
