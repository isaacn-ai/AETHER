# Titan Cosmetic Frontend

This repository is configured as a Vite + React + TypeScript storefront project for **Titan Cosmetic**.

## Run locally

```bash
npm install
npm run dev
```

## Validate and build

```bash
npm run lint
npm run build
npm run preview
```

## Repository state

This checkout currently includes project configuration and metadata files only. Application source files (`src/`), public media assets, and product/page content files are not present in this snapshot.
